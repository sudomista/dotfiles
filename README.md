# Dotfiles
Maked with soft kanagawa color palette

### Requirements
WM: ```sway``` (```swayidle``` & ```swaylock``` too)

Bar: ```waybar```

Menu: ```wofi```

Notify: ```mako```

Terminal: ```foot```

Shell: ```fish```

Fetch: ```fastfetch```

Screenshots: ```grim``` & ```slurp```

### Installing 
```bash
git clone https://github.com/sudomista/dotfiles
mv * ~/.config/
```

### Note
It's worth noting that I maked a configs in sense of my screen resolution and etc. In this case, you will need to manually adjust, for example, the size of the waybar or directories in some files.
